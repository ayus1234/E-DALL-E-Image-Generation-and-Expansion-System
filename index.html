E-DALL-E: Image Generation with Varying Aspect Ratios

This project combines text-to-image generation using DALL-E Mini with a novel expansion technique (E-DALL-E) to create images with different aspect ratios.

Overview

Modern AI image generation models like DALL-E Mini typically generate square (1:1) images. However, for many applications, different aspect ratios like 16:9 for landscapes or 9:16 for portraits are preferred. E-DALL-E solves this problem by expanding the generated square images to any desired aspect ratio.

How It Works

The process consists of two main steps:

  1. Image Generation: Generate a square image using DALL-E Mini based on your text prompt.
  2. Image Expansion: Expand the image to your desired aspect ratio using:
    • First, an initial expansion with repeated edge pixels
    • Then, optimization with VQGAN+CLIP to generate realistic content in the expanded areas

Getting Started

To use this project, you'll need to install the required dependencies:

pip install -r requirements.txt

Using the Command Line

Run the script with a text prompt to generate and expand an image:

python edalle.py --prompt "a painting of rolling farmland" --aspect_ratio 1.78 --num_images 4

Or use the simpler script to expand an existing image:

python simple_expand.py image.jpg --aspect 16/9 --prompt "a beautiful landscape" --output expanded.jpg

Using Google Colab

You can also try the project in Google Colab without installing anything locally. Open in Colab

Limitations

While E-DALL-E produces good results for many images, it has some limitations:

  • The quality of expansion depends on the original image content
  • Highly structured content (like faces) may be difficult to expand naturally
  • The optimization process can be computationally intensive
